# (G)ava Pre-trained Transformer

## How to use?

#### get repository
> git clone https://github.com/serezk4/JPT.git \

#### set-up your config at folder src/main/resources/...:
> application.properties\
> telegram.properties\
> gpt.properties

#### set-up environment, required
> JDK21\
> MySQL Server

#### make bootable jar
> gradle bootJar

### Run!

## How to update?
> git pull origin master (or) dev
